# 🏆 Module 02: Nobel Prize Practice Exercises

### 🎯 Overview & Learning Objectives
This module focuses on practicing intermediate SQL queries using the **Nobel Prize Dataset**. The exercises are designed to master data filtering techniques, relational operators, and conditional logic to extract specific insights from historical laureate data.

---

### 🛠️ Core SQL Concepts Covered
* **Basic Filtering:** Harnessing `WHERE` clauses to isolate data points based on specific criteria.
* **String Matching & Wildcards:** Using `LIKE` operators to scan names and specific patterns within text columns.
* **Logical & Comparison Operators:** Utilizing `>`, `<`, `=`, `AND`, `OR`, `NOT`, and `BETWEEN` to build multi-conditioned queries.
* **Special Operators:** Implementing `IN` to query matching lists and handling specific case exclusions.

* `nobel_queries.sql` - Complete SQL scripts with clean formatting and solutions for each quiz problem.
